## Usage

### Components

To start using components, import them into your project:

```tsx
import { Button } from "@szum-tech/design-system";

function App() {
  return <Button variant="primary">Click me</Button>;
}
```

### Icons

To start using Icons, import them into your project:

```tsx
import { GoogleLogoIcon } from "@szum-tech/design-system/icons";

function App() {
  return (
    <div>
      <GoogleLogoIcon />
    </div>
  );
}
```
